# 🌍 Project Name: INT250-GA2-007x-UIKIT

โปรเจคนี้พัฒนาขึ้นโดยใช้ HTML ร่วมกับ **UIkit Framework** เพื่อสร้างอินเทอร์เฟซที่ทันสมัยและเป็น Responsive

## 🚀 Getting Started (การเริ่มต้นใช้งาน)

นายสามารถเริ่มต้นพัฒนาโปรเจคนี้ได้ง่ายๆ ตามขั้นตอนด้านล่างนี้:

### 1. การติดตั้ง (Installation)
โปรเจคนี้ใช้ไฟล์ **UIkit แบบ Local** (โหลดลงเครื่องแล้ว) เพื่อให้สามารถพัฒนาแบบ Offline ได้:

* **CSS:** อยู่ในโฟลเดอร์ `/css/uikit.min.css`
* **JS:** อยู่ในโฟลเดอร์ `/js/uikit.min.js`
* **Icons:** อยู่ในโฟลเดอร์ `/js/uikit-icons.min.js`

### 2. วิธีการรันโปรเจค (How to Run)
1.  Clone repository นี้ลงเครื่อง:
    ```bash
    git clone https://github.com/UltraPixel042/int250-GA2-007x-UIkit.git
    ```
2.  เปิดไฟล์ `index.html` ผ่าน Browser หรือใช้ **Live Server** ใน VS Code เพื่อดูผลลัพธ์แบบ Real-time
